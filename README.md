# React-Redux, Redux-Saga

Demonstrate of a simple react application development using the following.

* React-Bootstrap
* React-Redux
* Redux
* Redux-Thunk
* Redux-Saga
* React-Router
* SASS

## Getting started
```shell
git clone https://github.com/khachornchit/Redux-Saga.git
cd Redux-Saga
npm run install
npm run start
```
